1. Git guarda las versiones únicamente en el equipo, mientras que GitHub guarda las versiones en la página.

2. Para Git solo se necesita un repositorio local (.git), mientras que para GitHub se necesita subir o crear un repositorio en la página; así como tener una cuenta de GitHub.

3. Git es para trabajar de forma autónoma, y GitHub se puede trabajar tanto de forma autónoma como colaborativa.
